# App Network Analyzer

## Overview
This project analyzes network request logs and maps them to user actions such as liking, commenting, or viewing content.

## Features
- Classifies API requests into:
  - LIKE
  - COMMENT
  - VIEW_POST
- Processes request logs from a file
- Generates summary counts of user actions
- Includes automated tests using pytest

## How to Run

```bash
python main.py
